# 🎧 Spotify Clone

A modern Spotify-inspired music streaming web app focused on clean UI, smooth interactions, and optimized performance.

---

## 🚀 Features

### 🏠 Pages Implemented

* **Home** – Main interface displaying songs
* **Sign Up** – User registration UI
* **Login** – Authentication interface
* **Favorites** – View and manage liked songs
* **Playlists** – Organize songs into collections

---

## ⚙️ Core Functionality

* 🎵 **Music Player**

  * Play / Pause
  * Track switching
  * Basic playback controls

* 🔍 **Search**

  * Real-time search functionality
  * Implemented **debounce** for better performance

* 🔁 **Toggle Features**

  * UI state toggling (e.g., play/pause, views)

---

## 🎨 UI & Experience

* Clean and responsive layout
* Spotify-inspired design
* Basic animations for smoother interaction

---

## 📂 Data Handling

* Songs are **static (locally managed)**
* No backend integration yet

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* DOM Manipulation
* Event Handling
